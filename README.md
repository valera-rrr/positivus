## О проекте

Positivus — одностраничный сайт, демонстрирующий навыки вёрстки и веб‑дизайна. Проект создан для:
* отработки навыков адаптивной вёрстки;
* практики работы с препроцессором SCSS;
* демонстрации понимания принципов UI/UX;
* закрепления знаний по семантической разметке.



## Технологии и инструменты

В проекте использованы:

* **HTML5**
    - семантическая разметка;
    - корректная структура документа.

* **SCSS (Sass)**
    - переменные (`_variables.scss`);
    - миксины;
    - вложенность селекторов;
    - частичные файлы (частички);
    - импорт компонентов.

* **CSS3**
    - Flexbox для компоновки;
    - медиа‑запросы (адаптивность);
    - кастомные свойства;
    - переходы и анимации.

* **Методология**
    - БЭМ‑нотация для классов;
    - модульный подход к SCSS;
    - разделение стилей по функциональности.

* **Дополнительные инструменты**
    - Google Fonts для типографики;
    - Font Awesome для иконок;
    - normalize.css для сброса стилей.

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/valera-rrr/positivus.git
   
2. Перейдите в директорию проекта:
   ```bash
   cd positivus
   
3. Запустите компиляцию SCSS в CSS:
   ```bash
   npx sass scss/main.scss css/style.css
   
4. Откройте файл `index.html` в любом современном браузере.
